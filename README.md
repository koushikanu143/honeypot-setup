\# Honeypot Setup



A beginner-friendly cybersecurity project that demonstrates how to build and run a basic honeypot using Python. This project is designed to simulate a vulnerable service, attract unauthorized login attempts, and record attacker activity for monitoring and learning purposes.



---



\## Project Overview



A honeypot is a decoy system used in cybersecurity to detect, observe, and analyze suspicious activity. In this project, a simple honeypot server is created to listen for incoming connections and log login attempts made by unknown users.



This project is useful for students, beginners, and internship demonstrations because it shows practical concepts such as:

\- network listening

\- fake service simulation

\- attacker interaction logging

\- basic security monitoring



---



\## Features



\- Simple Python-based honeypot

\- Monitors incoming connection attempts

\- Records fake login usernames and passwords

\- Saves logs in a text file

\- Easy to understand for beginners

\- Good for cybersecurity mini project demonstration

\- Python 3
